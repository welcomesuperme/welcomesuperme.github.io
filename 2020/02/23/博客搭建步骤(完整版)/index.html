博客搭建步骤(完整版)

Posted on 2020-02-23

第一步：安装Node.js 点击安装

第二步: 注册github用户 搜索github在官网进行注册

第三步:下载git软件 360搜索git软件下载并安装，安装时需注意其中一步需要选择Use Windows’ default console window。其他就直接点Next就OK了

第四步:配置git环境 。打开 安装好的git文件中的git -bash.exe 输入 ssh-keygen -t -C “邮箱”一直回车(但要记住key的路径,一会要用到,一般在第三行的()里),直到出现一个$停止。打开自己的github，把右上角的下拉框打开，单击settings,在左边找到SSH and GPG keys单击，右边上面那个框填写用户名,下面那个填写刚才记住的路径下的id_rsapub文件拷贝到下面那个框里,点击下面的那个ADD~按钮即可。

第五步安装hexo：搜索hexo在官网复制 npm install hexo-cli -g,在D盘建文件夹,我的是blog（根据个人喜好定义）window+R cmd 输入 D:回车，回车 输入cd blog(文件夹名称) 回车,输入官网复制的 npm install hexo-cli -g
进行安装(可能时间会长一点),安装完成后,回车,输入hexo init 博客名,回车进行安装,安装完成后,输入 cd 博客名，回车，输入nmp install 回车，进行安装npm，安装完毕后,回车,输入hexo s -p 5555,回车，此时该博客就可以在浏览器上打开，在浏览器输入（”http://localhost:5555/"）即可查看博客,（只是只能在本机访问,并且下次就不能访问了，主题和文本也都是系统默认的）。

第六步 安装sublime Text3:去浏览器搜索sublime Text3去官网安装即可。安装完成后,点击左上角文件（file）->打开文件夹->选中刚才的blog文件夹，此时点击source，posts下的文件里面就可以修改博客信息。

第七步 将hexo发布到github上去:点击_config.yml，滑到最下方找到type:它后面的东西删掉，后面写git，进入github找到仓库->点击Code->点击Clone or download->复制网址,在type下一行填写repo: 网址回车 ，输入branch: master，退出sublime Text3 window+R ->输入cmd回车->切换到blog 博客名的文件夹下输入 hexo g回车,输入npm install hexo-deployer-git –save回车进行安装,安装完成后,再输入hexo g回车，输入hexo d进行提交回车，输入git config –global user.email “邮箱地址”回车,输入git config –global user.name “用户名”，回车,输入用户名密码，hexo d 再进行提交。此时输入我们的网址便可查看博客，已经公开发布了。

第八步使用主题：百度搜索hexo主题（点击进入）,在下面的search框里搜索next
点击Next,就来到了github的源码，点击题目为Installation下的 detailed installation instructions里面有安装步骤但不详细,点击标题为option1下的 latest release version进行下载,下载要下载标题为Assets下的Source(zip),完成后进行压缩,把压缩后的文件剪切到blog/博客名/themes里面并改名为next,然后到sublime Text3下的_config.yml文件下找到关键字themes把他后面的内容修改成next（记得保存）。然后 window+R cmd 切换到blog/博客名文件夹下,输入hexo g 回车，输入命令行hexo s -p 5555，回车,此时本地就可看到next格式下的博客。在 sublime Text3的next文件夹下找到_config.yml文件,找到scheme:Muse将它注释掉,将它下面的scheme:Mist的注释去掉

更换主题（点击进入）
推到远端：hexo clean —> hexo d(文件需以md结尾)
